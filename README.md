# nl-ease
A tool to easily set the night light. Simple tool written in C and the Enlightenment Foundation Libraries. It allows you to choose the color intensity, set a start time, end time and save the settings. (nl = "night light" 😊 )

Designed for Bodhi Linux 7 but may also work on other distros with EFL development package installed.\
Supported languages: English, Italian and Polish.

## Requirements:

nl-ease requires an installation of EFL development package.

<ins>***Example EFL development package installs (Bodhi Linux 7):***</ins>
```
sudo apt update
sudo apt install libefl-dev
```
### Installation:

- **Compile:**  `make`

- **Install:**  `make install`

- **Uninstall:**  `make uninstall`
